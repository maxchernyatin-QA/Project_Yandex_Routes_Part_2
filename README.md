В рамках проекта была поставлена цель протестировать каршеринг веб-приложения "Яндекс.Маршруты".

Задачи:

1. Подготовить чек-лист и провести тестирование вёрстки.

2. Подготовить чек-лист и провести тестирование окон "Способ оплаты" и "Добавление карты".

3. Составить тест-кейсы и провести тестирование логики кнопки "Забронировать".

4. Составить баг-репорты.

Выполнено:

1. Подготовлен чек-лист из 55 проверок, выполнено тестирование, обнаружено 17 багов.

2. Подготовлен чек-лист из 43 проверок, выполнено тестирование, обнаружено 9 багов.

3. Составлены 5 тест-кейсов, выполнено тестирование, обнаружено 4 бага.

4. Составлены 27 баг-репортов.

Выделить классы эквивалентности и граничные значения.

Протестировать валидацию полей ввода времени и адресов в форме, логику расчёта стоимости и времени поездки на собственном автомобиле.

Составить баг-репорты по результатам проведенного тестирования.

Выполнено:

Проведен тест-анализ, выделены основные блоки для тестирования логики полей ввода.

Выделены классы эквивалентности и граничные значения.

Создано 70 тест-кейсов и проведено тестирование каждого из полей ввода.

Составлено 16 баг-репортов.
