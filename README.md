В рамках проекта была поставлена цель протестировать каршериенг веб-приложения "Яндекс.Маршруты"

Для реализации поставленной цели были выполнены следующие задачи:

1. Составлен чек-лист и проведено тестирование вёрстки веб-приложения "Яндекс. Маршруты"

2. Составлен чек-лист и проведено тестирование окон "Способ оплаты" и "Добавление карты" веб-приложения "Яндекс. Маршруты"

3.  Составлены тест-кейсы и проведено тестирование логики кнопки "Забронировать" веб-приложения "Яндекс. Маршруты"

4.  Составлены баг-репорты
